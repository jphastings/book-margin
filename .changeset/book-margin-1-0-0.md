Book Margin 1.0.0 — sync Kindle highlights and Highlighted.app exports into your
atproto repository as `at.margin.note` Web Annotations, reviewed and edited on a
static site before saving. Highlighted exports reference physical books via the
Margin physical-books spec (page numbers and ranges); Kindle clippings carry
their reading location. First published versions of the supporting libraries.
